Spirited Away Type Face

×
This was a challenging project that required me to use Adobe Illustrator to create an entire scene using text. The scene depicted is from Hayao Miyazaki's "Spirited Away" and it is created using important quotes from the movie. The goal of this project was to experiment with typography, composition and hierarchy to create an captivating illustration.

ACS Nano Cover Submission

×

As part of my ongoing work with the Blue Sky Initiative at the University of Michigan, I created 3D scenes to be used as cover designs for the team's submission to the ACS Nano journal. These images highlight the chirality (rotation) of a protein fiber, showcasing the complex structures and properties of the fibers in a visually striking way. My work was selected as a supplementary cover for ACS Applied Nano Materials!
