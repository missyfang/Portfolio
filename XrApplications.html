XR Applications

Here you can learn more about the augmented and virtual reality applications I have worked on.

FaceAR

FaceAR is an augmented reality app that helps patients with facial disabilities perform facial exercises accurately. The app uses face tracking technology to guide users through exercises and immersive games that make the experience more engaging. Check out the project site!

  • Learn more

A2Go

This Ann Arbor-themed version of PokemonGo is an augmented reality mobile game developed using Unity. It encourages players to explore the city of Ann Arbor, discover local artist's murals, and interact with the city's landmarks in a unique and engaging way.

  • Learn more

VR lab

This virtual reality recreation of a computer lab is built using Unreal Engine 5 and offers a variety of interactive features that enhance the user's experience. It allows users to explore a computer lab in a highly realistic and immersive way.
